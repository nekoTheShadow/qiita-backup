    パスカルの三角形をOracle SQLで実装した

    お仕事の休憩時間に頭の体操で作成したので、お炊き上げです(´・ω・`)

    パスカルの三角形をOracle SQLで実装しました。実用性は正直ないです。SQLでここまでできるとも言えますし、こんなSQLを書いてはいけませんともいえます。

    なお環境はOracle Database 19cになります。

    WITH
seq (n) AS (
    SELECT 1 FROM DUAL
  UNION ALL
    SELECT n+1 FROM seq WHERE n <= 10
),
triangle (x, vals) AS (
      SELECT 1, ',1,' FROM DUAL
    UNION ALL
      SELECT 
        x+1,
        (SELECT ',' || LISTAGG(
           CASE WHEN n=1 OR n=x+1 THEN
             '1'
           ELSE
             TO_CHAR(  TO_NUMBER(SUBSTR(vals, INSTR(vals, ',', 1, n-1)+1, INSTR(vals, ',', 1, n)  -INSTR(vals, ',', 1, n-1)-1))
                     + TO_NUMBER(SUBSTR(vals, INSTR(vals, ',', 1, n  )+1, INSTR(vals, ',', 1, n+1)-INSTR(vals, ',', 1, n  )-1)))
           END, ',') WITHIN GROUP(ORDER BY n) || ',' 
         FROM seq
         WHERE n <= x+1)
      FROM triangle
      WHERE x <= 10
)
SELECT SUBSTR(vals, 2, LENGTH(vals)-2) AS " Pascal's triangle" FROM triangle ORDER BY x;
