    WSL2からWindows Explorerを開きたい (Macのopenコマンドと同様のことをWSL2でも実現したい)

    問題

    Macのターミナルにはopenというコマンドがあって、たとえばopen .とするとカレントディレクトリがFinderで表示します。同じことをWSL2で実現したい場合はどのようにすればよいでしょうか? つまりWSL2からWindows Explorerを開きたい場合、どのようにすればよいでしょうか?

    解決策

    たとえばWSL2のカレントディレクトリをWindows Explorerで開きたい場合、解決策は次の2つになります。

    • cmd.exe /c start .
    • explorer.exe .

    どちらも覚えておくにはやや長いので、.bash_profileなどにエイリアス登録や関数定義しておくとよいでしょう。ちなみに自分は次のようなwindow関数なるものを.bash_profileに定義して使っています。

    window(){
  if [[ $# -eq 0 ]]; then
    cmd.exe /c start .
  else
    cmd.exe /c start $1
  fi
}

    環境情報

    • Winedows: Microsoft Windows [Version 10.0.19041.450]
    • WSL2: Ubuntu 20.04 LTS
