    ずんだの1桁足し算問題 Java11編

    tbpgrさんの「ずんだの1桁足し算問題 Ruby編」を2018年9月にリリースされたJava11で実装しました。

    import java.util.stream.IntStream;

public class Main {
    public static void main(String[] args) {    
        IntStream.range(1, 10)
            .boxed()
            .flatMap(x -> {
                return IntStream.range(1, 10).mapToObj(y -> {
                    int z = x + y;
                    var tilda = "~".repeat(z);
                    return String.format("%d+%d=%s%d", x, y, tilda, z);
                });
            })
            .forEach(System.out::println);
    }
}

    このコードのJava11らしい点といえば、Java11より導入されたString::repeatを利用していることでしょうか。「とある文字列をn回繰り替えした文字列を生成する」APIがJavaではながらく標準として用意されておらず、自分でUtilメソッドを作成したり、外部ライブラリを用意したりする必要がありましたが、Java11よりString::repeatが標準ライブラリに実装されています(参考: https://qiita.com/neko_the_shadow/items/78ad73e86f45e4994a89)
