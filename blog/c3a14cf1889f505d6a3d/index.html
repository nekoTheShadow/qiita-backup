    2つのテーブルのデータがすべて一致しているかを確認したい

    テーブルXとテーブルYのデータがすべて一致しているかを確認したい場合、どのようにすればよいでしょうか?

    集合Xと集合Yが等しいことを確認したい場合、以下の1または2のどちらかを確認すればよいです。

    1. $(X \subseteq Y) = (Y \subseteq X)$
    2. $(X \cap Y) = (X \cup Y)$

    これをそのままSQLにすればよいですね。

    1.については、以下のSQLが結果セットを返さない場合、テーブルXとテーブルYは等しいといえます。

    SELECT *
FROM ((SELECT * FROM x
       EXCEPT
       SELECT * FROM y)
       UNION ALL
      (SELECT * FROM y
       EXCEPT
       SELECT * FROM x)) AS z;

    2.についても、テーブルXとテーブルYが等しいなら、以下のSQLの結果セットは0件になります。

    SELECT *
FROM ((SELECT * FROM x
       UNION
       SELECT * FROM y)
       EXCEPT
      (SELECT * FROM x
       INTERSECT
       SELECT * FROM y)) AS z;
